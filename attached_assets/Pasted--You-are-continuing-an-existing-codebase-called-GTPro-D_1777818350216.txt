========================
CORE PRODUCT DEFINITION
========================

GTPro is an AI-powered trading intelligence system with 3 continuous bot fleets:

1. Security Bot Fleet (SBF)
2. Vulnerability Check Bot Fleet (VCBF)
3. Agent Bot Fleet (ABF)

All fleets must run continuously in the background (24/7 simulation mode is acceptable for now).

Users control ONLY:
- starting/stopping bot sessions
- selecting strategy
- setting duration
- enabling/disabling auto-execution mode

Bots NEVER stop thinking unless user stops session.

========================
1. AGENT BOT FLEET (ABF) — CORE ENGINE
========================

Implement ABF as a continuous async loop per active user session:

Every 15–60 seconds:
- Fetch simulated or real market snapshot
- Send data to AI model (nemotron-ocr-v1 or fallback simulation if unavailable)
- Receive:
  - signal (BUY / SELL / HOLD)
  - confidence score (0–100)
  - reasoning text
  - strategy alignment

Store each signal in database and stream to frontend via WebSocket.

ABF must:
- run per user session
- support multiple bots (one per linked account)
- never block UI
- always log decisions
